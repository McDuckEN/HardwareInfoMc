What is a Keyboard ?


In the past, when there were no computers, typewriters were used to write the required content. In general, it can be said that a computer keyboard or computer keyboard is a set of buttons that you can type your desired text using mechanical levers.


In a computer keyboard, each key is assigned a specific ASCII code that types a specific letter. If you want to type specific letters with the keyboard, you need to press two or more keys on the computer keyboard, which should be done more about Persian letters.


Keyboard is used as a computer peripheral hardware that has different types. With the advancement of knowledge and technology, various keyboard models have entered the market, each of which has specific features and is designed for specific people.


What is the use of a computer keyboard ?


Computer keyboards are used in different situations due to their different types.


The main use of a computer keyboard is to act as a human-computer interface for writing letters, numbers, and symbols to word processing software and programs similar to word processing programs.


Another use of a computer keyboard is to apply a series of defined and functional commands, for example, as soon as you press the Ctrl + Alt + Delete keys together, the task manager page will open for you.


These combined buttons can send predefined commands to the computer. In this way, you can use the keyboard to perform operations when you do not have access to the mouse.




Types of keyboards :


Since the invention of the keyboard, the keyboard has made significant progress and various models of this tool have been introduced to the market.


The difference between keyboards is in their dimensions, appearance and most importantly their use. Each keyboard in the market today is made for a specific use and a specific group of people, and today each class of people can buy a specific type of keyboard according to their use of the computer.
