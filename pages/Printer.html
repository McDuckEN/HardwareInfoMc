What is a Printer ?



A printer is a hardware output device that is used to produce a hard copy and print any document. A document can be of any type such as a text file, an image or a combination of both.


It accepts input commands from users on a computer or other devices to print documents.


For example, if you have to submit a project report for your university, you need to create a software version of your report and print it with the help of a printer.



Printers are one of the most common computer peripherals, divided into two categories: 2D and 3D printers. 2D printers are used to print text and graphics on paper, and 3D printers are used to create three-dimensional physical objects.


The printed output produced by a printer is often called a hard copy, which is the physical copy of an electronic document. While some printers can only print black and white hard copies, most printers today can produce color prints. In fact, many home printers can now produce quality photos that rival professionally designed photos. This is because modern printers have high DPI (dots per inch) settings, which allow documents to be printed with very good resolution.


Most modern printers connect using a standard USB cable. However, some printers can be connected wirelessly to one or more computers via a Wi-Fi network. You can also use more than one printer on a single computer, as long as the correct drivers are installed.
